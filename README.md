# drone-flight-checklist-v2
This project aims to change the concept of form checking for drone from manual form to digital form. The form must be dynamic because there are differences in each drone that will be used.

# SETUP
1. Extract Project Folder "drone-flight-checklist-Web.zip"
2. Move Folder "drone-flight-checklist' to "C:\xampp\htdocs\" #!!! MUST DO !!!#

# XAMPP Setup
1. Open Folder "drone-flight-checklist"
2. Find "drone_flight_check.sql" in folder "C:\xampp\htdocs\drone-flight-checklist\class\database"
3. XAMPP / localhost:
    1. START Apache
    2. START MySQL
    3. Open "http://localhost/phpmyadmin/"
    4. Create Tabel Name: "drone_flight_check"
    5. Import Tabel "drone_flight_check" from folder "C:\xampp\htdocs\drone-flight-checklist\class\database"

# WEB
1. Open Link in New Tab: "http://localhost/drone-flight-checklist/" to open Login Page
2. Login or Register Account
3. If done Registering Account, go login with Username and Password that is registered
